# LtrustRx – Enterprise GenAI Security Testing Platform

LtrustRx is an enterprise-style GenAI Security Testing Platform designed to simulate and analyze AI/LLM security risks including Prompt Injection, Jailbreak attacks, RAG exploitation, API abuse, data leakage, and healthcare AI security threats.

---

## Features

- Prompt Injection Testing
- LLM Jailbreak Simulation
- AI Red Team Payload Testing
- RAG Security Simulation
- API Security Checks
- SSRF/XSS Payload Testing
- Healthcare AI Security Analytics
- Severity-Based Risk Scoring
- Enterprise Dashboard UI
- Admin Scan History
- Dynamic Category Filtering
- AI Threat Simulation Engine

- FastAPI backend routes
- AI security scanner engine
